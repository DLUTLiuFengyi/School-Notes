### case

* 硬币面额

  ```c
  // dp[i] 表示面额为i最少需要的硬币数
  // 初始
  dp[i] = MAX_VALUE
  // 状态转移
  dp[i] = min(dp[i], dp[i-codes[j]] + 1)
  ```

* 机器人走方格

  ```c
  // 初始
  dp[i][j] = 1 (i=0 || j=0)
  // 状态转移
  dp[i][j] = dp[i-1][j] + dp[i][j-1]
  ```

* 青蛙跳石头

  能不能跳到

  ```c
  // dp[j] 表示青蛙能不能跳到石头j
  dp[j] = OR(f[i] AND i+a[i]>=j) (0<=i<j)
// 初始
  dp[0] = true
  dp[j] = false
  ```
